Andrew Lister's Tech Transfer Pages

I am an afficionado of sampling statistics, remote sensing, and all things forest monitoring. I have provided the below tools in case people who work in these areas might find them useful. Please let me know (andylister100 at gmail.com) if you have any suggestions, comments, or find errors! NOTE: As of 2023, I am slowly transitioning my content to this new site, so stand by and I will be adding more of my tools and demos.

Tool Description
Confidence Interval Demo A spreadsheet tool that gives an intuitive sense of how confidence intervals relate to sampling variability.
CV Explorer A tool that allows users who plan inventories to assess the impacts of sample CV, required precision (E%, as a percentage of the estimate) and confidence level desired for a confidence interval. It is in English, Spanish and French.
Covariance Demo A simple demonstration of the relationship between the variance of an estimate from calculating per-plot sum or difference values and then calculating their variance vs. calculating that from calculating the totals of 2 different pools and including the covariance.
Rapid Photointerpretation Method Demo MS Access file and Python code for setting up a rapid photointerpretation project in MS Access, using ArcMap to prepare files. It's basically an offline, primordial version of Collect Earth (https://www.collect.earth/)
Plot design simulation R scripts This code accompanies the plot design simulation and optimization paper found at https://www.fs.usda.gov/research/treesearch/65591. It can be used to test plot design optimality under different stand and landscape spatial heterogeneity scenarios.
Collect Earth Online analysis tools. These tools are for the analysis of Collect Earth Online (CEO)-style data to make estimates for land cover/use status and trends. The point of CEO is to create a probability sample-based estimate of land cover/use class status or trends, using high resolution imagery and ocular photointerpretation. For more information on CEO, see https://www.collect.earth/
